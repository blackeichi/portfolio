# ✨ FE Developer (4년차) – 한정우

## 소개

안녕하세요. 대용량 데이터 처리 환경에서의 렌더링 성능 최적화와 사용자 경험 개선을 강점으로 가진 프론트엔드 개발자 한정우입니다.

대량의 데이터를 다루는 테이블에서 virtualization과 렌더링 구조 개선을 통해 초기 렌더링 시간과 성능 최적화를 개선한 경험이 있으며, React Profiler를 기반으로 병목 구간을 분석하고 불필요한 렌더링을 줄이는 최적화 작업을 수행해왔습니다.

또한 실시간 데이터 대시보드 개발 과정에서 상태 관리 구조를 개선하여 데이터 갱신 시 발생하던 렌더링 비용을 줄이고, 사용자 인터랙션이 많은 환경에서도 안정적인 성능을 유지하도록 설계했습니다.

단순한 기능 구현을 넘어, 성능과 구조 개선을 통해 실제 사용자 경험을 개선하는 개발을 지향합니다.
