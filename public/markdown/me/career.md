### 3. 듀얼아이 DMPW 시스템 리뉴얼

**기간**: 2024.01 ~ 2024.03

**프로젝트 개요**
회사 메인 상품의 v2 리뉴얼 프로젝트

**주요 성과**

#### 아키텍처 개선

- **클래스형 컴포넌트 → 함수형 컴포넌트 전환**
- 커스텀 훅을 만들어 **로딩/요청/에러/재시도 로직 공통화**
- 전역 변수, 팝업창 등 생명주기 관리 로직 추가

#### 성능 최적화

- **Intersection Observer를 활용한 무한 스크롤 구현**
  - 데이터가 화면에 근접할 때만 순차적으로 로딩하도록 최적화
- 메모이제이션, 책임 기준의 컴포넌트 세분화를 통한 리렌더링 최소화
- **사용자 인터랙션 시 발생하던 렉 이슈 완전 해결 및 성능 대폭 향상**
- 코드 분할(dynamic import), 이미지 최적화, CLS 최소화

#### 코드 품질 개선

- 동일한 역할을 하는 서로 다른 라이브러리 통일
- 불필요한 코드 삭제 및 흩어진 코드 정리 리팩토링
- 오래된 라이브러리를 최신 버전으로 교체하여 **보안성 향상 및 보일러플레이트 최소화**

#### UI/UX 개선

- 트렌드에 맞는 UI 적용
- 폰트 적용 및 전체적인 색상 통일화로 가시성 향상
- 레이아웃 정리를 통한 불필요한 여백 최소화 및 직관성 향상
- **사용자 친화적 UX 구현**
  - 테이블 버튼 고정, 툴팁, 헤더 메뉴 숨기기(인터랙션)
  - 데이터 로딩 전 로딩 표시
  - 불필요한 반복 동작 줄이기
  - 자연스러운 애니메이션
