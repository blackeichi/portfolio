# 📝 앱 소개

> 식단, 루틴, 일기, 투두를 한 곳에서 관리할 수 있도록 만든 올인원 라이프 관리 서비스입니다.  
> 웹과 모바일(WebView) 환경을 함께 고려해 설계했고, 인증 흐름, 서버 상태 관리, 초기 로딩 경험, 성능 최적화를 중심으로 구조를 지속적으로 개선해왔습니다.

- **배포 주소**: https://daily-log-frontend-piof.vercel.app/
- **형태**: 모노레포 기반 웹 / 모바일 연동 프로젝트
- **Frontend**: Next.js 15, React 19, TypeScript, Tailwind CSS, React Query, Jotai
- **Backend**: NestJS, Prisma, PostgreSQL(Neon)
- **Mobile**: Capacitor(Android WebView)

---

### 페이지 소개

- **홈 대시보드**: 하루 기록과 주요 상태를 한눈에 확인
- **식단 관리**: 식단 / 칼로리 기록 조회 및 관리
- **루틴 관리**: 반복 루틴 생성, 체크, 순서 관리
- **로그 작성**: 하루 기록 작성 및 조회
- **투두 관리**: 해야 할 일 등록, 완료 처리, 순서 변경
- **프로필 관리**: 계정 정보 및 비밀번호 변경
